# AIFitnessCoach

## Stack
- Expo + React Native + TypeScript
- Supabase (PostgreSQL)
- Groq LLM

## Comandos
npm start | npm run ios | npm run android

## Estructura
app/(tabs)/ - Pantallas principales
components/ - Contextos y componentes
hooks/ - Hooks personalizados
config.js - API keys desde .env
